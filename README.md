This project is a base for building new projects in PHP.

## Requirements

-   PHP 7.2 or later
-   MySQL or MariaDB
-   Apache Server

## Installation

You can install this project via composer with the following commands:

```bash
$ composer install
```

## Configure the project

-   Copy the [`.env.example`](./.env.example)
    file and call it `.env`.

```bash
$ cp .env.example .env
```
